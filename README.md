# Dictionary-BE
backend of the dictionary

Setting up for the expressjs to work
<h3>Create new server</h3>

<p>    mkdir server</p>
 <p>cd server</p>
 <p> npm init -y</p>
  <p>  npm install nodemon -D </p>

<h3>install the express js</h3>

<p>    cd server </p>
<p>   npm install express </p>
 <p>   npm run dev </p>
